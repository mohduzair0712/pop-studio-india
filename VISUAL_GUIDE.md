## ✨ ANIMATION EFFECTS

```
1. Page Load:
   Content fades in from bottom ↑
   Duration: 1 second
   Timing: Staggered timing

2. Scroll Animations:
   Cards appear with slide-up effect
   Triggers when in viewport
   Smooth easing function

3. Hover Effects:
   All cards lift up 5-10px
   Box shadow enhances
   Smooth transition (0.3s)

4. Button Hover:
   Move up 2px
   More shadow
   Slight scaling

5. Counter Animation:
   Numbers increment smoothly
   When section comes in view
   Duration: 2 seconds

6. Parallax:
   Background elements move slowly
   On page scroll
   Creates depth effect
```

```

Mobile:   20px padding on each side

Vertical Spacing:
Section gaps: 80px between sections
Card gaps: 2rem (32px) between cards
Element gaps: 1-1.5rem within cards

Grid Layouts:
- 3 columns desktop → 2 columns tablet → 1 column mobile
- 4 columns desktop → 2 columns tablet → 1 column mobile
- Responsive auto-fit

Hero Title:     3.5rem (desktop) → 2.2rem (tablet) → 1.8rem (mobile)
```
